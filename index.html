Sistema Electrónico basado en microprocesadores y lógica programable, diseñados para realizar pocas funciones específicas.

Los sistemas embebidos vinculados al concepto de IoT (Internet of Things), se diseñan para tener conectivadad e interactuar con la web.

El microcontrolador es un chip que tiene los mismos componentes que una computadora. Micropocesador, memoria, interfaces de comunicación, etc. Su finalidad es ejecutar código de programa que realizan una tareas específica.

El FPGA es un chip que tiene una serie de componentes lógicos que no se encuentran interconectados. Dicha conexión se realiza por medio de un lenguaje de descripción de hardware (HDL).

Muy utilizado en Inteligencia Artificial.

Misión

Sitio web de sistemas embebidos con información relevante que sirva para llevar adelante proyecto profesionales.

Visión

Ser un espacio de referencia de buenas prácticas sobre diseño e implementación de sistemas embebidos.

Valores

Respetar a quienes aportan el conocimiento y a las organizaciones que financian actvidades de I+D.
