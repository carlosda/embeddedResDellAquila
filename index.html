¿Qué es un SISTEMA EMBEBIDO?

Sistema electrónico digital con capacidad de procesamiento de datos, diseñados para realizar pocas funciones específicas.

¿Dónde hay SISTEMAS EMBEBIDOS?

Los sistemas electrónicos embebidos se encuentran en diversos sistemas y dispositivos. Se encargan de realizar tareas de cálculo, control, procesar eventos de entradas (un botón, un sensor) y comandar actuadores para generar salidas.

¿Existen diferentes tipos de SISTEMAS EMBEBIDOS?

De acuerdo a la cantidad de información que deben procesar y si es una aplicación crítica, más complejo es el sistema embebido. Hay sistema que no pueden dejar de funcionar bajo ninguna condición, por ejemplo, un equipo médico.

IoT y los SISTEMAS EMBEBIDOS

Internet de las Cosas (IoT) incluye a dispositivos simples, tal como sensores que miden nuestro entorno, que se encuentran conectados a internet. Los SISTEMAS EMBEBIDOS son muy importantes para su desarrollo.
